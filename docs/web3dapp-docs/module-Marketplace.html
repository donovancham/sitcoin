(async, inner) grantAllowance(amount)

Grants allowance from the user to the NFTMarket contract. The user must have enough $SITC in his wallet balance before the grant can be fulfilled. The NFTMarket contract is allowed to make $SITC transfers from the user's account to the seller's account if the user has granted an allowance.

Name Type Description
amount Number
